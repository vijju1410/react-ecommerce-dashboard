# React E-Commerce Dashboard

This project is a React-based e-commerce dashboard built for an internship assignment.

Features:
- User registration and login
- Session expiry after 5 minutes
- Protected routes
- Product listing using FakeStore API
- Add to cart functionality
- User-specific cart storage
- Profile update
- SweetAlert notifications

Tech Stack:
- React (Create React App)
- React Router
- Tailwind CSS
- LocalStorage
